# My Reselling Site

A full-stack reselling business management app built with Next.js 14, TypeScript, Tailwind CSS, and SQLite.

## Features
- 📊 Dashboard with sales stats and low-stock alerts
- 📦 Inventory management (add, edit, delete products)
- 💰 Sales recording with profit tracking
- 🔍 Search and filter products and sales

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Seed Data

```bash
npm run seed
```

## Build

```bash
npm run build
npm start
```

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- better-sqlite3 (SQLite)
